# SkyBrain 城市低空物流无人机集群协同调度与风险规避平台

## 📋 使用手册

### 🎯 平台概述

SkyBrain是面向城市低空经济的无人机集群智能调度与风险规避平台，基于动态贝叶斯网络和多智能体协同决策技术，为城市低空物流提供全方位的智能化解决方案。

### 🎮 功能模块详解

#### 1. 📊 实时监控模块
**功能描述**: 提供无人机集群的实时状态监控和关键指标展示

**主要功能**:
- 实时无人机状态监控（位置、电量、任务状态）
- 系统性能指标展示（CPU使用率、内存占用、网络延迟）
- 任务执行统计（完成率、平均时间、成功率）
- 告警信息实时推送

**操作指南**:
1. 点击"📊 实时监控"标签页
2. 查看左侧3D场景中的无人机实时位置
3. 右侧面板显示详细的数据统计
4. 红色指标表示异常，需要关注

#### 2. 🧠 风险评估模块
**功能描述**: 基于动态贝叶斯网络的多维度风险评估系统

**核心算法**:
- **天气风险**: 风速、降雨、能见度、温度
- **障碍物风险**: 建筑物、电线、移动障碍
- **空域风险**: 其他飞行器、禁飞区域
- **设备风险**: 电池状态、传感器健康度

**操作指南**:
1. 选择"🧠 风险评估"标签页
2. 查看实时风险概率矩阵
3. 点击"重新评估"按钮更新风险数据
4. 红色区域表示高风险，需要避开

**风险等级说明**:
- 🟢 **低风险** (0-30%): 安全飞行
- 🟡 **中风险** (30-70%): 谨慎飞行
- 🔴 **高风险** (70-100%): 禁止飞行

#### 3. 🤖 多智能体协同模块
**功能描述**: 基于分布式共识算法的无人机集群协同决策

**核心技术**:
- **共识算法**: PBFT (实用拜占庭容错)
- **任务分配**: 匈牙利算法优化
- **冲突解决**: 优先级调度机制
- **通信协议**: 自定义轻量级协议

**操作指南**:
1. 进入"🤖 多智能体"模块
2. 查看智能体状态矩阵
3. 点击"启动协商"开始任务分配
4. 观察共识达成过程

#### 4. 🛣️ 路径规划模块
**功能描述**: 动态A*算法结合实时避障的智能路径规划

**算法特点**:
- **动态A***: 实时重规划能力
- **多目标优化**: 距离、时间、安全性
- **实时避障**: 动态障碍物检测
- **能耗优化**: 考虑电池续航

**操作指南**:
1. 选择"🛣️ 路径规划"标签页
2. 设置起点和终点坐标
3. 选择优化目标（最短距离/最短时间/最安全）
4. 点击"开始规划"生成路径
5. 蓝色线条显示推荐路径

#### 5. 🔗 多模态感知模块
**功能描述**: 融合多种传感器数据的环境感知系统

**传感器类型**:
- **视觉传感器**: RGB摄像头、红外摄像头
- **激光雷达**: 3D点云数据
- **气象传感器**: 风速、湿度、气压
- **通信传感器**: 5G/WiFi信号强度
- **GPS传感器**: 高精度定位
- **IMU传感器**: 姿态和加速度

**数据融合算法**:
- **卡尔曼滤波**: 状态估计
- **粒子滤波**: 非线性系统
- **贝叶斯融合**: 不确定性处理

#### 6. 🚁 集群调度模块
**功能描述**: 智能化的无人机集群任务调度与资源管理

**调度策略**:
- **优先级调度**: 紧急任务优先
- **负载均衡**: 避免单机过载
- **动态调整**: 实时任务重分配
- **故障转移**: 自动备机切换

**操作指南**:
1. 进入"🚁 集群调度"模块
2. 查看当前任务队列
3. 可手动分配任务或使用自动调度
4. 监控任务执行进度

#### 7. ⚠️ 风险预警模块
**功能描述**: 实时风险监控与智能预警系统

**预警类型**:
- **设备故障预警**: 电池、传感器、通信
- **环境风险预警**: 恶劣天气、障碍物
- **空域冲突预警**: 其他飞行器接近
- **任务异常预警**: 超时、偏航、失联

**预警级别**:
- 🟢 **信息**: 一般提醒
- 🟡 **警告**: 需要关注
- 🟠 **严重**: 立即处理
- 🔴 **紧急**: 自动处置

#### 8. 💼 商业管理模块
**功能描述**: 运营数据分析与商业智能报告

**核心功能**:
- **收入分析**: 实时收入统计和趋势分析
- **客户管理**: 客户信息、订单历史、满意度
- **性能指标**: KPI监控、效率分析
- **报告导出**: JSON/CSV格式数据导出

**报告类型**:
- **日报**: 当日运营概况
- **周报**: 一周业务总结
- **月报**: 月度经营分析
- **季报**: 季度战略报告
