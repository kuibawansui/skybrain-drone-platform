# SkyBrain 城市低空物流无人机集群协同调度与风险规避平台

## Core Features

- 动态贝叶斯网络风险图谱

- 多智能体协同决策

- 多模态感知融合

- 3D城市仿真系统

- 实时风险热力图

- 商业化管理平台

## Design

深色科技主题，以深蓝色为主色调配合亮蓝色强调，融合航空航天专业感与现代科技未来感，采用发光效果、渐变和半透明元素，模块化布局支持多屏实时监控，包含3D仿真主视区、实时数据面板、风险热力图等核心功能区域
